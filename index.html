Sezione 1: L'Espansione Ricorsiva Polinomiale (La Discesa)

Il nucleo fondamentale del framework NOSEC risiede nel rifiuto dell'approccio crittanalitico tradizionale. I metodi classici di fattorizzazione tentano di aggredire il modulo intero $N$ considerandolo come un blocco monolitico e statico; questo approccio sbatte inevitabilmente contro il muro della complessità computazionale esponenziale non appena la lunghezza del numero supera i parametri standard.

NOSEC scardina questo limite imponendo una trasformazione dinamica: il problema della fattorizzazione viene convertito in una struttura geometrico-frattale auto-similare, costringendo il modulo originario a frammentarsi in sottoproblemi progressivamente più piccoli attraverso una discesa ricorsiva condizionata.

The core foundation of the NOSEC framework lies in the rejection of traditional cryptanalytic approaches. Classical factorization methods attempt to attack the integer modulus $N$ by treating it as a monolithic, static block; this approach inevitably hits the wall of exponential computational complexity as soon as the bit-length exceeds standard parameters.

NOSEC shatters this limitation by enforcing a dynamic transformation: the factorization problem is converted into a self-similar geometric-fractal structure, forcing the original modulus to fragment into progressively smaller subproblems through a conditioned recursive descent.

L'Equazione Pilota e la Genesi del Residuo

L'algoritmo avvia la scansione partendo da una coordinata di base $x$ arbitraria, geometricamente prossima alla radice quadrata del modulo da fattorizzare ($x \approx \sqrt{N}$). The algorithm initiates scanning from an arbitrary base coordinate $x$, geometrically close to the square root of the target modulus ($x \approx \sqrt{N}$). The fundamental relationship between the modulus $N$, the chosen base $x$, and the unbalance coefficient $b$ is defined by the second-degree polynomial equation:

$$N = x^2 + bx + c$$

In questa formulazione, il coefficiente di sbilanciamento intero $b$ rappresenta lo scostamento lineare ed è calcolato deterministicamente tramite l'operatore di parte intera:

In this formulation, the integer unbalance coefficient $b$ represents the linear offset and is deterministically calculated via the floor operator:

$$b = \left\lfloor \frac{N - x^2}{x} \right\rfloor$$

The key element of the descent is the local residue $c$, meaning the computational area portion that exceeds the square and the unbalance rectangle:

$$c = N - x^2 - bx$$

Per le proprietà intrinseche della divisione euclidea e della scelta della base, il residuo $c$ è soggetto a un vincolo matematico rigido e inviolabile: esso è strettamente minore della base di calcolo del livello corrente. Al primo livello di discesa, abbiamo la certezza assoluta che:

Due to the intrinsic properties of Euclidean division and base selection, the residue $c$ is bound to a strict, inviolable mathematical constraint: it is strictly smaller than the current level's calculation base. At the first descent level, we possess absolute certainty that:

$$c < \sqrt{N}$$

Lo Sviluppo Telescopico Frattale

Il valore $c$ diviene il modulo radice di un nuovo sottoproblema indipendente. Il sistema genera così un albero di discesa profondo $n$ livelli, dove ogni residuo intermedio $c_i$ viene frammentato utilizzando una nuova base locale $x_i$ e un nuovo coefficiente $b_i$:

The value $c$ becomes the root modulus of a new independent subproblem. The system thus generates a descent tree $n$ levels deep, where each intermediate residue $c_i$ is fragmented using a new local base $x_i$ and a new coefficient $b_i$:

  • Level 1: $N = x^2 + bx + c_1 \quad (\text{with } c_1 < \sqrt{N})$
  • Level 2: $c_1 = x_1^2 + b_1 x_1 + c_2 \quad (\text{with } c_2 < \sqrt{c_1})$
  • Level 3: $c_2 = x_2^2 + b_2 x_2 + c_3 \quad (\text{with } c_3 < \sqrt{c_2})$

Sostituendo iterativamente ogni livello, si ottiene l'Espansione Telescopica Globale di $N$:

By iteratively substituting each level, we obtain the Global Telescopic Expansion of $N$:

$$N = x^2 + bx + x_1^2 + b_1 x_1 + x_2^2 + b_2 x_2 + \dots + x_n^2 + b_n x_n + c_{n+1}$$

Sezione 2: La Risalita per Step Deterministici (Filtrazione Combinatoria)

Se la fase di discesa opera per scomposizione geometrica, la fase di risalita agisce tramite un algoritmo di filtrazione deterministica per step successivi, eliminando la necessità di effettuare tentativi probabilistici.

While the descent phase operates via geometric decomposition, the ascent phase works through a step-by-step deterministic filtration algorithm, completely eliminating the need for probabilistic attempts.

Step 1: La Fattorizzazione Esaustiva del Residuo Foglia

Al livello foglia $n$, il residuo $c_n$ è estremamente piccolo. Il singolo nodo della rete distribuita esegue una fattorizzazione esaustiva locale per determinare l'insieme completo e finito di tutti i divisori di $c_n$, mappando istantaneamente tutte le $D$ coppie possibili tali che:

At leaf level $n$, the residue $c_n$ is extremely small. The single distributed network node executes an exhaustive local factorization to find the complete, finite set of all divisors of $c_n$, mapping all $D$ possible pairs such that:

$$x_{n,1} \cdot x_{n,2} = c_n$$

Step 2: Instantaneous Linear Filtration

Affinché una coppia di divisori sia algebricamente valida per iniziare la risalita, essa deve soddisfare in modo esatto il vincolo lineare imposto dal coefficiente $b_n$ calcolato durante la discesa:

For a divisor pair to be algebraically valid to begin the ascent, it must precisely satisfy the linear constraint imposed by the coefficient $b_n$ computed during the descent:

$$x_{n,1} + x_{n,2} = b_n$$

La Congruenza di Coerenza Globale

L'intero sistema di filtrazione, dal livello foglia fino al modulo radice $N$, è governato da una singola congruenza fondamentale rispetto al fattore primo cercato $p$:

The entire filtration system, from the leaf level up to the root modulus $N$, is governed by a single fundamental congruence relative to the sought prime factor $p$:

$$\sum_{i=0}^{n} x_i(x_i + b_i) \equiv 0 \pmod p$$

Sezione 3: Architettura di Rete e Calcolo Distribuito Parallelo

Lo spazio delle coordinate iniziali $x$ viene segmentato dal server in intervalli discreti denominati Work Units ($\Delta X = [x_{\text{start}}, x_{\text{end}}]$). I nodi client analizzano le unità in modo asincrono, eseguendo la discesa e la filtrazione locale dei divisori a zero-overhead di banda, comunicando i risultati rari solo in caso di superamento dei filtri strutturali di risalita.

The initial coordinate space $x$ is segmented by the server into discrete intervals called Work Units ($\Delta X = [x_{\text{start}}, x_{\text{end}}]$). Client nodes analyze these units asynchronously, performing descent and local divisor filtration with zero bandwidth overhead, reporting rare matches only when structural ascent filters are passed.
